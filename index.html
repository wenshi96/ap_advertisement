How Would an Elected Board Work

  • Our proposal for an elected board would create twenty districts across the city. Elections would start in 2023, and each district would elect one member every four years. The president of the board would be elected city-wide. With this system of both local and city-wide representatives, we hope the new board will be able to reflect concerns of neighborhoods while also reflecting the sentiments of the city. The larger board will also allow for more local representation with grassroot campaigning and make it easier to hold the board accountable.
